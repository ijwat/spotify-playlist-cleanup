# Spotify Playlist Cleanup

## Overview

This project provides a Python script that cleans up Spotify playlists by removing duplicate tracks. It utilizes the Spotify Web API via the Spotipy library to interact with user playlists.

## Features

- Fetches user's playlists and displays the total number of tracks.
- Identifies and removes duplicate tracks based on their unique URIs.
- Provides feedback in the terminal about the number of duplicate tracks found and removed.
